# Developer-only helper scripts

These are commit-and-push convenience wrappers that **must not** be part of
the normal launch path. They were originally littering the repo root as
`JARVIS_*PUSH*.bat` / `JARVIS_GIT_PUSH.ps1`; they are kept here for the
maintainer's muscle memory but are not invoked by any user-facing flow.

If you don't recognize what they do, you don't need them. Use the supported
launchers under `scripts/jarvis/` instead.
